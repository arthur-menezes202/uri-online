# Beecrowd

Bem vindo ao meu repositorio.

Este é o repositório onde guardo minhas submissões do site beecrowd, fique a vontade para navegar e caso queira tirar alguma dúvida sobre a lógica de cada exercício.

# Regras

Não é permitido copiar as respostas
- Caso queira, é permitido comparar ou entender a lógica por trás de cada exercício, mas o código deve ser unicamente seu.
- Organização

# Organização

O repositório está organizado da seguinte maneira, no site do beecrowd, existem as categorias de cada exercício, e cada exercício tem uma numeração, o repositorio possui uma Pull request para cada categoria e a numeração de cada exercício, vale lembrar que não possui todos as categorias e exercícios, conforme vou resolvendo elas vão aparecendo.
</br>
</br>
Dentro de cada PR, averá uma pasta com o nome da linguagem, e cada uma tera seus respectivos exercícios
